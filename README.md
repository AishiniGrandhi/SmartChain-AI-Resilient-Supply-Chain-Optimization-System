# 🛰️ SmartChain AI Control Tower

An enterprise-grade, multi-domain operational intelligence platform for resilient supply chain management. This system utilizes AI-driven predictive analytics and high-fidelity network topology mapping to optimize global logistics.

---

## 🚀 Getting Started

### 1. Prerequisites
- Any modern web browser.
- **Optional**: Gemini API Key for live AI responses (add to `script.js`).

### 2. How to Run
Simply open **`index.html`** or **`dashboard.html`** in your web browser. No terminal or installation is required.

---

## 🧠 Key Features

- **Multi-Domain Intelligence**: Adaptive UI for Logistics, Aviation, Maritime, and Emergency Response.
- **Interactive Network Topology**: High-fidelity map synchronization with animated telemetry markers.
- **AI Strategic Assistant**: Collapsible FAB for real-time strategic queries (Gemini 1.5 integrated).
- **Asset Management**: Full module for registering and searching operational assets.
- **Mission Control UX**: Professional glassmorphism design with persistent navigation and actionable alerts.

---

## 🛠️ Technology Stack
- **Languages**: HTML5, CSS3, ES6+ JavaScript.
- **Libraries**: Leaflet.js (Mapping), Chart.js (Visualization), FontAwesome 6 (Icons).
- **AI**: Google Gemini 1.5 Flash API.
