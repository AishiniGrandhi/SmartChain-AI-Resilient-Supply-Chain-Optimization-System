# SmartChain AI Control Tower

### Intelligent Supply Chain Resiliency & Predictive Optimization

Modern global supply chains manage millions of concurrent shipments across highly complex and inherently volatile transportation networks. Critical transit disruptions ranging from sudden weather events to hidden operational bottlenecks are chronically identified only after delivery timelines are already compromised.

**SmartChain AI** is a scalable system designed to continuously analyze multifaceted transit data to preemptively detect and flag potential supply chain disruptions. It formulates dynamic mechanisms that instantly execute or recommend highly optimized route adjustments before localized bottlenecks cascade into broader delays.

---

## 🚀 Objective

To design a scalable, AI-driven control tower capable of:
1.  **Preemptive Detection**: Analyzing real-time transit data to identify risks before they manifest.
2.  **Dynamic Optimization**: Formulating instant route adjustments to bypass bottlenecks.
3.  **Strategic Insights**: Leveraging Large Language Models (LLMs) to provide actionable logistics intelligence.

## ✨ Key Features

-   **Glassmorphism Control Tower**: A premium, high-density dashboard for global visibility.
-   **Gemini AI Strategy Hub**: Integrated AI assistant for real-time strategic decision-making and risk analysis.
-   **Predictive Demand Engine**: Advanced data visualization (Chart.js) comparing actual vs. forecasted demand patterns.
-   **Dynamic Transit Map**: Real-time shipment tracking on interactive Leaflet maps with animated transit routes.
-   **Intelligent Inventory Management**: Automated stock-level monitoring with color-coded depletion alerts.
-   **Unified Alert Center**: Centralized hub for critical, warning, and informational system notifications.

## 🛠️ Tech Stack

-   **Frontend**: Vanilla HTML5, CSS3 (Custom Glassmorphism Design System), JavaScript (ES6+).
-   **Visualization**: Chart.js for data analytics.
-   **Mapping**: Leaflet.js with custom dark-matter tile layers.
-   **AI Engine**: Google Gemini 1.5 Flash API integration.
-   **Icons & Fonts**: FontAwesome 6, Google Fonts (Outfit & Inter).

## 📦 Installation & Usage

1.  Clone the repository.
2.  Open `index.html` in any modern web browser.
3.  Navigate to the **AI Strategic Hub** and paste your [Gemini API Key](https://aistudio.google.com/app/apikey) in `script.js` to enable predictive insights.
